How it works

The robot observes its position within the current step, its velocity, whether it's grounded, and remaining stamina — all relative to the current stair, so one compact policy generalises to every step. It chooses among IDLE, WALK, JUMP, and JUMP RIGHT. Weak air control means it must build a run-up before leaping, and solid risers block walking through — so it has to learn to time a momentum-assisted hop at each stair. Random play only reaches the top ~55% of the time; a trained agent ≈100%.

Source & the original Python version are in the repository. Built with Q-learning, no frameworks.
