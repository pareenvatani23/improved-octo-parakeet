How it works

Each floor is a perfect maze of corridors; floors are linked by vertical shafts. The robot's state is simply which cell it's in (floor, row, column), and it picks among six moves — north, south, east, west, up, down. It only learns from reward: −1 for every step and +100 for reaching the exit. Through Q-learning it gradually builds a map of which move is best from each cell, converging on the shortest escape. Random wandering almost never escapes within the move budget; a trained agent escapes every time.

Self-contained isometric renderer + tabular Q-learning. No frameworks.
